# 什么是流程控制

简单来说，就是**控制程序的执行顺序**。对于流程控制，可以分为三种结构，分别是：

•顺序结构

•分支结构

•循环结构



# 顺序结构

顺序结构，指的是js程序**默认的流程结构**，按照书写顺序从上至下执

# 条件语句

## iF语句

~~~js
var name = 1
if (name == 1)
{
    console.log("233")
}
~~~

## if...else语句

~~~js
var name =1;
if(name == 1)
{
    console.log("nice")
}
else
{
    console.log("啊哈哈")
}
~~~
